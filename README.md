# yesafs
This is a simple FUSE filesystem that just forwards operations (open, read, and write) to the underlying filesystem.

### Instruction
- Compile : ```./bin/compile.sh```
- Mount : ```./bin/mount.sh```
- Unmount : ```./bin/unmount.sh```
